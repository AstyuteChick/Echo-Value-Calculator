Concept Overview

NOTE: In the next articles, certain sections are enclosed in square brackets “[-]” and are in italics. Readers may skip these sections without loss of continuity and/or understanding. They are only there to over-explain content, add more justifications or caveats, or provide extra information.

Note for first-time readers: This section is best skimmed. This is an overview, big-picture explanation and is intended as something to refer back to as each individual concept is explained.

Here is the main conceptual formula:

\[\mathbf{ES} = \frac{AV}{PV}*100 = \frac{\sum_{i}^{}\left( \frac{{SV}_{i}}{{SA}_{i}}*{RV}_{i} \right)}{\sum_{i = 1}^{5}{ORV}_{i}}*100,\]

where:

  • \(ES\) = Echo Score

  • \(AV\) = Achieved Value / Total Relative Value of Average Substat rolls present on an Echo under consideration*

  • \(PV\) = Potential Value / Total Relative Value of the Current Best 5 Average Substat Rolls*

  • \(SV\) = Substat Roll (the actual number (value) displayed on the Echoes)

  • \(SA\) = Substat Average (the average value of this substat)

  • \(RV\) = Relative value of the average of this substat (not the stat) compared to other substats (dynamic for ER!)

  • \(ORV\) = Ordered Relative value (arranged in descending order, from highest to lowest)

*This is technically incomplete still, as the values are also weighted and effective, but this is automatic for all stats except ER with the method used.

The Build Score is simply the average Echo Score of the five Echoes of a set, however, each Echo processed may have an impact on the next Echo’s achieved value and potential value during a full analysis of a build, increasing the accuracy of the scores obtained.

After clarifying and justifying these terms, anybody should be able to calculate the scores using a pen, paper, and a (normal) calculator (albeit with significantly more time and effort than just using the EVC).
